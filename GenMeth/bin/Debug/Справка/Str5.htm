Коллекция отношений

После завершения работы с коллекциями имён и свойств можно приступить к работе с редактором отношений. Для этого служит пункт меню главного окна приложения "Проект\Редактор отношений".

В случае успешного открытия редактора отношений в нём появится пустой список отношений. Для добавления отношений служит кнопка панели инструментов "Добавить отношение".

В левом списке появившегося диалогового окна перечисляются первичные ключи, в правом - внешние ключи. Для установления отношения требуется выбрать первичный ключ и соответствующий ему внешний ключ. После этого необходимо ввести имя отношения, псевдоним отношения, имя ограничения и псевдоним ограничения. После чего выбрать правило для удаления и правило для обновления.

Приложение автоматически проверяет соблюдение правил ввода идентификаторов и их уникальность. В случае нарушения этих правил выводится соответствующее сообщение.

Когда все параметры установлены, необходимо кликнуть кнопку "Принять", Отношение добавится в список.

При дальнейшем добавлении отношений внешние ключи, для которых уже установлены отношения в правом списке не отражаются. Когда отношения установлены для всех внешних ключей, добавление отношений становится невозможным, но остаётся возможность редактирования отношений.

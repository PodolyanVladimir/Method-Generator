Как работать с программой

Работа с программой состоит из четырёх этапов:

  1. Создание коллекции имён таблиц и столбцов.
  2. Создание коллекции свойств столбцов.
  3. Создание коллекции реляционных отношений.
  4. Генерация программного кода метода.

Каждый последующий этап подразумевает полное выполнение предыдущего.

Незавершённость какой-то коллекции не позволит перейти к следующим этапам.

Назначение программы

Программное приложение "Генератор методов создания объектов локального представления данных" предназначен для программистов, работающих с использованием языка программирования C#.

Данный программный продукт освобождает программиста от рутинного труда по написанию больших объёмов программного кода.

Для работы с программой не требуется особых навыков.

Для использования результатов работы программы, а именно, сгенерированного программного кода, необходимо знание языка C# и наличие навыков проектирования в интегрированных средах разработки, таких, как SharpDevelop или VisualStudio.

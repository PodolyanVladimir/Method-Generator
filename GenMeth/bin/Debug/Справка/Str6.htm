Генерация программного кода метода

Генерация программного кода метода является конечным этапом работы с приложением. Для получения возможности генерации кода должны быть окончательно сформированы все коллекции: коллекция имён, коллекция свойств и коллекция отношений.

Генерация кода вызывается из пункта меню главного окна приложения "Проект\Выполнить генерацию метода".

Сгенерированный код метода может быть скопирован в буфер обмена или сохранён в текстовом файле.

Содержание сгенерированного кода состоит из трёх частей:

  1. пространства имён;
  2. объявления имён объектов;
  3. описание метода создания объектов локального представления данных.

Каждая из частей должна быть правильно размещена в проекте разработчика. Пространства имён подключаются перед пространством имён проекта. Объявления имён объектов лучше расположить перед конструктором класса. Описание самого метода размещается после конструктора класса.
